== Installation ==

1. **Download & Upload** - Download the plugin and upload the `advanced-image-gallery` folder to `/wp-content/plugins/`
2. **Activate** - Go to Plugins → Installed Plugins and activate "Advanced Image Gallery for Elementor"
3. **Add to Page** - In Elementor, drag the "Advanced Image Gallery" widget to your page
4. **Customize** - Choose your layout, add images/videos, and customize settings
5. **Publish** - Save and view your stunning gallery!

**Quick Start:**
1. Add the widget to your page
2. Select Grid, Carousel, or Slideshow layout
3. Upload your images or add video URLs
4. Customize spacing, navigation, and effects
5. Publish and enjoy!

== Frequently Asked Questions ==

= How do I create a gallery? =
Simply drag the "Advanced Image Gallery" widget from Elementor's widget panel onto your page. Choose your preferred layout (Grid, Carousel, or Slideshow), add your images or videos, and customize the settings to match your design.

= Can I use videos in my gallery? =
Yes! The plugin supports both YouTube and Vimeo videos. Just paste the video URL in the video field, and the plugin will automatically detect and embed it with a custom play button overlay.

= Is the plugin mobile-friendly? =
Absolutely! All galleries are fully responsive and look perfect on desktop, tablet, and mobile devices. You can even set different column counts for different screen sizes.

= Do I need Elementor Pro? =
No! The plugin works with both Elementor Free and Elementor Pro. However, some advanced features like custom CSS may require Elementor Pro.

= Can I customize the appearance? =
Yes! The plugin offers extensive customization options including colors, spacing, navigation styles, overlay effects, and more. Everything can be styled to match your brand.

= Will it slow down my website? =
No! The plugin is optimized for performance with conditional loading. Swiper.js and other assets only load when needed, keeping your site fast and lightweight.

= Can I use it for e-commerce? =
Yes! The plugin is perfect for product galleries, portfolio showcases, and any visual content display. It works great with WooCommerce and other e-commerce solutions.

= Is there a limit on the number of images? =
No! You can add as many images as you want. The plugin is designed to handle large galleries efficiently.

= Can I add links to images? =
Yes! You can link images to custom URLs, media files, or use the lightbox feature for full-screen viewing.

= Does it work with page builders other than Elementor? =
Currently, the plugin is designed specifically for Elementor. However, it may work with other page builders that support Elementor widgets.
